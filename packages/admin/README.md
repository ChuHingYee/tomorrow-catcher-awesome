# @tomorrow-catcher-awesome/admin

the admin-platform for tomorrow-catcher

## Getting started

```bash
# clone el proyecto
git clone https://github.com/ChuHingYee/tomorrow-catcher-awesome.git

# install dependency
pnpm i

# develop
pnpm --filter admin dev
```

## Build

```bash
# build
pnpm --filter admin build
```
